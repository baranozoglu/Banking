# Banking
<h1>Technologies to use</h1>   

● Java 11+    
● SpringBoot    
● MyBatis    
● Gradle    
● Postgres    
● RabbitMQ    
● JUnit   

<h1>Run</h1>
Follow this steps to run application.


`./gradlew build`  
`docker build . -t banking`   
`docker-compose up`

<h3>Now Listening</h3>
`127.0.0.1:8080/swagger-ui.html`
